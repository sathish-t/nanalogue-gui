# demo/showboat-notes/

Narrative demos of CLI tools, generated by
[Showboat](https://github.com/simonw/showboat). Each file walks through
a feature with real shell commands and their captured output.

## Adding a new demo

Run Showboat against the CLI you want to document and drop the resulting
`.md` file here. Do not hand-edit files in this directory — they are
treated as Showboat output.
