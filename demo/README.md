# demo/

Demo data, screenshots, and automation scripts used for showcasing and
testing the app.

## Contents

The BAM, FASTA, BED, JSON, and PNG files at the top level are demo data
and committed screenshots used by the scripts below and in documentation.

### Scripts

| File | What it does |
|---|---|
| `generate-demo-bam.mjs` | Generates the demo BAM, FASTA, and BED files from scratch (run once) |
| `take-screenshots.mjs` | Launches the app via Playwright and takes a full set of product screenshots; run via `npx playwright test --config demo/playwright.config.mjs` |
| `playwright.config.mjs` | Playwright config for `take-screenshots.mjs` |

### Screenshots

Committed screenshots used in documentation and the GitHub repository.
Regenerate them by running `take-screenshots.mjs`.

### Subdirectories

| Directory | What it is |
|---|---|
| [`rodney-demo/`](rodney-demo/README.md) | Shell script that drives the landing page via rodney (Chrome DevTools Protocol) and a companion CI workflow |
| [`showboat-notes/`](showboat-notes/README.md) | Narrative demos of CLI tools, generated by Showboat |
